TL;DR — Key Concepts
  • COBIT 2019: 5 domains — EDM, APO, BAI, DSS, MEA
  • NIST CSF 2.0: Added GOVERN as 6th function (Feb 2024); supply chain moved from Identify to Govern
  • SABSA: Sherwood Applied Business Security Architecture; 6 questions x 6 layers; risk-driven
  • ITIL: SLA management; 5 core publications, 26 processes; independent review every 3 years
  • Password Types: Static (min security) → Complex → Passphrase → Cognitive → OTP (max security)
  • Controls: Managerial (admin), Operational, Technical (logical); types include directive, deterrent, preventive, corrective
  • SSAE-18: Replaced SAS 70; SOC 1 Type 2 = design + effectiveness over time
  • Compliance = LAW: Non-negotiable; must be done first. Don't muddy the waters!
  • Continuous Monitoring: Must capture operational baselines first to detect abnormal patterns
  • Certification vs Accreditation: Certification = technical eval; Accreditation = overall security adequacy
