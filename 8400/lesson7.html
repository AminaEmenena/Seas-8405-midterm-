TL;DR — Key Concepts
  • TOCTOU = race condition; countermeasure is atomic instructions
  • Code Review: Formal (line-by-line) vs. Lightweight (pair, email, tool-assisted)
  • Session Management: desktop, remote, cloud, VLAN sessions
  • REST = Representational State Transfer; no specific message format required
  • Firmware vs Operating Ware: firmware is proprietary (EPROM); operating ware is meant to be changed
  • DevSecOps: bakes security in vs. slapping it on at the end
  • NIST SP 800-163: guideline (not standard) for mobile app vetting; 5 guidelines
  • Containerization: encrypted container separating corporate from personal data
  • COPE > BYOD: corporate-owned gives more control
  • SDLC Security Gate: if insecure at any phase, STOP
